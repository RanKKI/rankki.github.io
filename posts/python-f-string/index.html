注意事项

可以在字符串中使用各种引号，只要保证和外部的引号不重复即可。

以下使用方式都是没问题的：

1
2
3
4
5
6
7
8

>>> f"{'Python'}"
'Python'
>>> f'{"Python"}'
'Python'
>>> f"""Python"""
'Python'
>>> f'''Python'''
'Python'

那如果字符串内部的引号和外部的引号相同时呢？那就需要 \ 进行转义：

1
2

>>> f"You are very \"handsome\""
'You are very "handsome"'
括号的处理
1
2
3
4
5

>>> f"{{74}}"
'{74}'

>>> f"{{{74}}}"
'{74}'

可以看出，使用三个括号包裹效果一样。

反斜杠
1
2
3
4

>>> f"You are very \"handsome\""
'You are very "handsome"'
>>> f"{You are very \"handsome\"}"
  File "<stdin>", line 1
SyntaxError: f-string expression part cannot include a backslash

你可以先在变量里处理好待转义的字符，然后在表达式中引用变量：

1
2
3

>>> name = '"handsome"'
>>> f'{name}'
'"handsome"'
